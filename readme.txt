== Description ==

This plugin allows you to place the FAQ system to your website.
Users of your website can send you a question and you can answer.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Simple FAQ To The Website'
3. Activate the plugin from your Plugins page.

= From WordPress.org =

1. Download 'Simple FAQ To The Website'.
2. Upload the 'Simple FAQ To The Website' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate 'Simple FAQ To The Website' from your Plugins page.
